Al enviar una petición post desde Postman a la ruta http://localhost:3000/registro, con el siguiente JSON

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde de manera exitosa, indicando que los datos han sido captados

{
    "estado": "Datos recibidos",
    "nombre": "Maria",
    "mensaje": "Hola comunidad"
}
